# SecureCloudGroup Homebrew tap

Install [SmartBrain](https://github.com/SecureCloudGroup/SmartBrain_3000):

```sh
brew install --cask securecloudgroup/tap/smartbrain
```

SmartBrain is a fully local, single-user AI assistant you run in Docker on your own machine. It needs
Docker (Desktop, Colima, OrbStack, or Engine). Open it from your Applications folder; first run
downloads the app image and opens it in your browser.

The cask is generated from [`packaging/homebrew`](https://github.com/SecureCloudGroup/SmartBrain_3000/tree/main/packaging) in the main repo.
